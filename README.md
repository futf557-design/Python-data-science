# 🌧️ Rainfall Distribution Analyzer

This project is a Python-based interactive data dashboard designed to analyze and visualize rainfall distribution across various districts in Tamil Nadu. It features a live data editor, dynamic charting, and a machine learning trend predictor.

## ✨ Features
* **Live Data Editing:** Edit, add, or delete rainfall records directly from the frontend UI.
* **Dynamic Visualizations:** Automatically updates Bar, Line, and Pie charts using `Seaborn` and `Matplotlib` based on user selections.
* **Machine Learning Prediction:** Uses `Scikit-Learn` (Linear Regression) to calculate and visualize rainfall trends over time.
* **Local Data Integration:** Uses Python's `os` module to seamlessly load the CSV dataset from the local project directory.
* **Cloud Deployment Ready:** Includes standard configuration files for seamless hosting on Streamlit Community Cloud.
